# Security Policy

Archon is an agent orchestration platform; we take security reports seriously
and respond in a timely manner.

## Reporting a Vulnerability

Please report suspected security issues by email to **security@trustin.fr**.

Include:

- A description of the issue and its potential impact
- Steps to reproduce (proof-of-concept repo or commands if possible)
- The affected version / commit SHA
- Your disclosure preferences (credit, embargo window)

We acknowledge reports within **3 business days** and aim to ship a fix or a
mitigation within **30 days** for confirmed vulnerabilities. Please do not
open a public GitHub issue for security-sensitive reports.

## Supported Versions

Security fixes are shipped on the latest minor release. Older majors are not
backported unless the fix applies to an active deployment.

## Disclosure

We follow a **coordinated vulnerability disclosure** process: once a fix is
released we credit the reporter (unless they prefer to stay anonymous) and
publish a GHSA advisory on the repository.
